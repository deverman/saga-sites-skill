# Landing Page Template Notes

Use the minimal-site template and keep the first screen as the actual product or
offer. Put page copy in `content/pages/home.md`; keep Swift renderers structural.

Recommended additions:

- one home renderer
- one contact or signup CTA
- Open Graph image
- style-guide page if a reusable component system is introduced
- `swift test`, `swift build`, and `saga build` checks
